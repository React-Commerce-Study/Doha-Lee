## 📁 Information

멋쟁이사자처럼에서 제공하는 오픈마켓 API으로 실제 오픈마켓 서비스처럼 만들어 본 오픈마켓 서비스입니다. 오픈마켓 서비스의 주요 기능은 다음과 같습니다.

- 로그인
- 회원가입
- 상품목록화
- 상품 상세페이지
- 장바구니
- 주문/결제하기
